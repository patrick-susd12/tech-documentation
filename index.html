Deprovisioning Chromebooks

Follow these steps to deprovision Chromebooks using the Google Admin console:

  1. Log in to Google Admin Console using SUSD credentials. Once logged in, click on Devices:
  2. Click on Chrome Devices:
  3. Change the Status from Provisioned to All, and then enter the device's serial number to find it in the console:
  4. Click on Deprovision from the menu options on the left panel of the console:
  5. Click on Yes, Factory Reset, then click the radio button next to Same Model Replacement, and check the checkbox to acknowledge the upgrade policy, then click the Deprovision button:

Make a Windows User an Admin on their Laptop

Using CMD to set current user as Admin on their district-issue laptop does not work on the G7 laptops. Here is a working method:

  1. Press the Windows Logo key & R key at the same time to open the Run dialog.
  2. Enter netplwiz in the text field and press Enter or click OK.
  3. A window will open with settings for user accounts. Click on Add...
  4. A dialog will appear to add a domain account. Click on Browse...
  5. In the Object Names dialog, type the user's SUSD username (first part of their email, with the susd12.org.
  6. Click on Check Names and make sure that the user's full name and SUSD email address shows up.
  7. Click on OK; once the dialog closes click on Next.
  8. Click the radio button that says Administrator, then click Next
  9. Click Finish and then click Apply, and OK
  10. Restart the laptop, and have the user log in to their account. The user will now have full admin control over their laptop.

    11. **NOTE: If a UAC dialog pops up asking the user to provide admin credentials, they can enter their SUSD username and password, and they will be granted admin priviledges.

Set up Secure Print for a User

  1. Once the printer is added to the user's computer, open up Control Panel and navigate to Devices and Printers.
  2. Locate the printer that the user wants to enable Secure Print on and righ-click and click Printing Preferences.
  3. A dialog box will appear. In the Basic Settings tab, click on the dropdown for Output Method:
  4. Click on the Secured Print option:
  5. An alert will pop up asking if you want to set the user's PIN now. Click on No to require the user to create a new PIN each time they wish to print a document. This will help combat users forgetting their Secure Print PIN:
  6. The user will now be set up for Secure Print on that printer, and will be prompted to provide a PIN each time they print a document to that printer.

Add LAMP Apps to SPED iPads in JAMF

  1. Log in to JAMF Dashboard.
  2. Go to Devices and look for the iPad by serial number.
  3. Move the iPad to the main Sunnyside OU (DO NOT move device owner!)
  4. Add device group SPED no apps
  5. Add device groupd SPED Device Group
  6. LAMP will install automatically.
  7. If TouchChat is needed, this will need to be manually installed from Managed Apps.
  8. At the top of the device profil ein JAMF, click on Permanently delete deffered items'.
